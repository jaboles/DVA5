DVA 5 Build Instructions

The following instructions have been tested on a fresh Mac OSX 10.11 virtual machine.

Prerequisites

  • The easiest way to install pre-requisite software tools is by using Homebrew, a package manager for Mac OSX. 
    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  • Java Development Kit version 1.6 or higher. 
    brew cask install java
  • Apache Ant. 
    brew install ant
  • Building a Windows setup requires the following additional tools:
    • Wine, required for running Inno Setup. Wine requires XQuartz. 
      brew cask install xquartz
      brew install wine
    • Inno Setup. Any recent version 5 should work. 
      curl -OL http://www.jrsoftware.org/download.php/is.exe && wine is.exe /silent

Building

  • Build DVA: 
    ./b.sh
    Build outputs will be placed in build/Debug, which will contain the built JAR (DVA.jar) as well as the Mac application package (DVA.app).
